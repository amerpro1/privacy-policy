Privacy Policy

MultiBox does not collect, store, or share any personal data.

However, third-party services such as Google AdMob may collect anonymous data for advertising purposes.

⚠️ All permissions are requested and used only after user approval.

Permissions

  • Camera: Used only when the user activates features such as QR scanning.
  • File Access: Used only for opening and saving files selected by the user.
  • Internet: Used only for ads.
  • Advertising ID: Used for ads.

All app features operate locally on the user's device.

The app uses largeHeap for better performance.

🔐 سياسة الخصوصية

لا يقوم تطبيق MultiBox بجمع أو تخزين أو مشاركة أي بيانات شخصية.

قد تقوم خدمات خارجية مثل Google AdMob بجمع بيانات غير شخصية لأغراض الإعلانات.

⚠️ جميع الصلاحيات يتم طلبها واستخدامها فقط بعد موافقة المستخدم.

الصلاحيات:

  • الكاميرا: تُستخدم فقط عند تشغيل ميزات مثل مسح QR.
  • الوصول للملفات: يتم فقط عند اختيار المستخدم لفتح أو حفظ ملف.
  • الإنترنت: يُستخدم فقط للإعلانات.
  • معرف الإعلانات: يُستخدم للإعلانات.

جميع ميزات التطبيق تعمل محلياً داخل الجهاز.

يستخدم التطبيق تحسين الذاكرة لضمان الأداء.
