Privacy Policy

This privacy policy applies to the MultiBox application.

Information Collection

MultiBox does NOT collect, store, or share any personal data from users. All features operate locally on the user's device.

Permissions Usage

All permissions are requested and used only after user approval.

  • Camera: Used only when the user activates features like QR scanning.
  • File Access: Used only for opening and saving files selected by the user.
  • Internet: Used only for displaying ads.
  • Advertising ID: Used by Google AdMob for ads.

Third-Party Services

The app uses Google AdMob to display advertisements. These services may collect anonymous data for advertising purposes.

Data Security

We do not collect any personal data, and no user data is stored or transmitted.

Children’s Privacy

This app does not knowingly collect personal data from children.

Changes to This Policy

We may update this policy from time to time. Updates will be posted on this page.

Contact Us
